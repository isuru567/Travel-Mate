# Travel Mate - Adventure & Trip Planning Platform

Travel Mate is a dynamic web application designed to help travelers discover destinations and plan their trips efficiently. This project was developed during my Industrial Internship, emphasizing clean UI components, diverse layouts, and a user-centric experience.

## Key Features
* **Template-Based Architecture:** Designed as a versatile website template that can be easily customized for different travel brands.
* **Multiple UI Variants:** Each core page features **three distinct UI layouts**, providing diverse visual options and enhanced user experience testing.
* **Destination Discovery:** Explore various travel spots with high-quality visual galleries.
* **Modern Typography:** Custom font integration for a unique and professional brand identity.
* **Responsive Layout:** Optimized for seamless performance on mobile and desktop browsers.
* **Organized Resources:** Structured asset management for fast rendering and better performance.

## Technologies Used
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP
* **Resources:** Custom Fonts and optimized image assets.

## Project Structure
* `resources/fonts/` - Custom typography files.
* `resources/images/` - Optimized destination and UI images.
* `Home/` & other modules - Core website functionality with multiple UI versions.
